# 左邓粉丝 · 银河综艺馆 · 同担座位互助

这是从单页原型拆出来的正式网站工程骨架。

## 当前状态
- 使用 Next.js 项目结构。
- 当前仍使用 mock 数据，方便先开发页面和流程。
- Supabase schema 已放在 `supabase/schema.sql`。
- 后续填入 `.env.local` 后接真实数据库。

## 本地运行
```bash
npm install
npm run dev
```
